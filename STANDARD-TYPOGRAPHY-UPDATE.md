# HOME31 V7.7.7 Standard Typography Update

This release keeps the V7.7.5 Executive Command Centre structure with eight equal KPI cards.

Changes:

- Standard display is now the default.
- A new display-settings storage key resets older Comfortable-mode preferences once.
- IBM Plex Sans is used consistently for body text, headings, KPI values and controls.
- Heading, KPI, sidebar, button and form sizes are normalized.
- Decorative serif figures are removed.
- The four-column executive KPI layout is retained on wide screens, with two columns on narrower screens and one column on phones.

No database or Edge Function change is required.
